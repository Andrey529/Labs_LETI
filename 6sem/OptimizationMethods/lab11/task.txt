Задание: написать функцию

trustreg(f, df,x0, tol)

реализующую поиск методом доверительных областей с:
1) использованием квазьньютоновской матрицы вместо матрицы Гессе (формула БФГШ для неинвертированной аппроксимации Гессиана)
2) с использованием алгоритма Dogleg для поиска внутри доверительной области
3) с использованием метода золотого сечения как локального поиска в алгоритме Dogleg

Входные переменные:
f - целевая функция,
df - производная целевой функции,
x0 - стартовая точка,
tol - заданная точность.

Выходная переменная:

answer_ = [xmin, fmin, neval, coords, radii]

где:

xmin - точка минимума,
fmin - значение целевой функции в точке минимума,
neval - число вычислений производной целевой функции,
coords - массив координат x во время оптимизации,
radii - массив радиусов доверительной области.

Целевые функции и их градиенты заданы аналитически.
В качестве критерия продолжения поиска используйте

(norm(dx) >= tol) and (k < kmax)

где kmax = 1000 - максимальное число итераций, dx - разница между предыдущим и текущим значениями x.
Для формирования массива coords и radii воспользуйтесь конструкцией

coords.append(x)
radii.append(r)

Величина константы для принятия решения о выборе нового шага
eta = 0.1

Для транспонирования используйте функцию transpose()
